# 마이 어리싱 (My Everything) — 모바일 MVP

개인 비공개 기반 라이프 기록 · 회고 · 자기이해 앱 (MVP)입니다.

## 핵심 원칙
- 기록/대화/설정은 **기기 안에만 저장**됩니다(로컬 저장).
- **SNS/공개/비교 기능 없음**.
- “답을 주는 AI”가 아니라 **질문으로 생각을 정리**하도록 돕는 톤.
- 치료·의료 표현을 직접 사용하지 않습니다.

## 필수 화면(요구사항 충족)
- Splash: 앱 이름/슬로건 표시
- 시작: 이메일 또는 익명 시작 + “완전 비공개” 안내
- 대시보드: 오늘 날짜 + (오늘의 기록/회고 보기/AI 조력자) 버튼
- 기록: 텍스트 입력/저장 + 저장 시간 표시
- 정서 요약: 안정/피곤/불안/혼란 중 하나로 요약 + 짧은 설명
- AI 조력자: 판단·비난 없는 질문 중심 대화(오프라인)
- 회고: 월간 리포트 “이번 달, 나의 이야기” 자동 문단 생성
- 설정: 비공개 안내 + 전체 데이터 삭제

## 로컬 실행 방법
프로젝트 루트에서 `mobile` 폴더로 이동해 실행합니다.

```bash
cd mobile
npm install
npm run start
```

### Android에서 실행
- Android Studio 에뮬레이터를 켠 뒤:

```bash
cd mobile
npm run android
```

### iOS에서 실행
- macOS가 필요합니다. (또는 Expo Go 앱을 사용해 미리보기)

## 데이터 위치
- `AsyncStorage`에 저장됩니다.
- 설정 화면에서 **전체 데이터 삭제**를 누르면 전부 삭제됩니다.
